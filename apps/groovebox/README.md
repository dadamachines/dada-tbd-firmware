# Groovebox

16-track drum machine and sequencer for TBD-16.

**Author:** [Per-Olov Jernberg (@possan)](https://github.com/possan)
**Repository:** [possan/tbd-pico-seq3](https://github.com/possan/tbd-pico-seq3)
**License:** MIT

## Features

- 16 drum tracks with per-step parameter locks
- Ableton Link synchronization
- MIDI controller support (Arturia BeatStep Pro, Launchpad)
- Pattern chaining and song mode
- Sample loading from SD card
