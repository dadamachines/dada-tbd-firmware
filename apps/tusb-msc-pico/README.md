# USB Mass Storage (Pico)

Exposes the RP2350 SD card as a USB mass storage device. Use this to transfer files to/from the Pico SD card via USB.

**Author:** [dadamachines](https://github.com/dadamachines)
**License:** MIT

## Usage

1. Boot into `tusb_msc` via the bootloader menu or flash directly in BOOTSEL mode
2. The RP2350 SD card will appear as a USB drive on your computer
3. Copy files to/from the drive
4. Eject properly before rebooting
